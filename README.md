# SQL Project: Product Performance Analysis

## Project Overview

This project simulates a real-world Product Performance Analysis scenario using SQL. The objective was to explore, clean, transform, and analyze sales data to generate business insights and support data-driven decision-making.

The project follows a modern data architecture approach using Bronze, Silver, and Gold layers, allowing data to move from raw ingestion through transformation into reporting-ready datasets.

## Tools & Technologies

* SQL Server
* T-SQL
* Common Table Expressions (CTEs)
* Window Functions
* Data Modelling
* Star Schema Design
* Business Intelligence Concepts

## Project Objectives

* Explore and understand database structures and relationships.
* Perform data cleaning and validation.
* Build analytical queries using joins, aggregations, and subqueries.
* Analyse product, customer, category, and regional performance.
* Generate business insights and recommendations.

## Database Structure

The project includes:

### Bronze Layer

Raw source tables containing:

* Products
* Categories
* Customers
* Regions
* Sales Transactions

### Silver Layer

Cleaned and transformed datasets with:

* Data quality validation
* Standardized values
* Business rule implementation

### Gold Layer

Analytics-ready star schema including:

* Fact Sales
* Dim Product
* Dim Category
* Dim Customer
* Dim Region

## Business Questions Solved

* Which products generate the highest revenue?
* Which categories contribute most to total sales?
* Which regions are underperforming?
* Who are the top-performing customers?
* Which products should receive additional marketing focus?
* How does revenue performance compare across periods?

## Key Insights

* Identified top-performing products contributing the largest share of revenue.
* Highlighted underperforming regions requiring business attention.
* Ranked customers based on revenue contribution.
* Analysed category profitability and sales trends.
* Measured product contribution percentages to overall revenue.

## Business Recommendations

1. Increase marketing investment for high-performing products.
2. Investigate underperforming regions and develop targeted sales strategies.
3. Focus customer retention efforts on high-value customers.
4. Review low-performing products and optimise inventory planning.
